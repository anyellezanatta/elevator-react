# Elevator React

This is a Vite React project that simulates the behavior of an elevator.
The elevator logic is fully implements using the hooks.

## Stack

- React 18
- Functional components
- Hooks
- Modern CSS
- Jest
- Stylelint
- ESlint
- Prettier

## Running

```bash
yarn dev
```

Follow the instuctions on the terminal.

## Linting

```bash
yarn lint
yarn lint:css
```

## Formatting

```bash
yarn format
```

## Testing

```bash
yarn test
```
